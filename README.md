# Get started

**Step by step:**

- `npm install`
- `cp .env.example .env`
- Edit `.env`
- `npm run start:dev`

**Production build:**

- `npm run build`
- `npm run start:prod`

**Docker compose:**

launch
- `docker-compose up -d`

rebuild and launch
- `docker-compose up -d --build`
